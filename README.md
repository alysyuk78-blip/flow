# Flow — проєкт-менеджер

Локальний веб-застосунок для ведення задач і проєктів. Поєднує найкращі ідеї
**Things**, **Notion**, **Todoist** та **Gantt Pro**.

Працює у браузері, без сервера. Дані зберігаються в localStorage. Українською.
Можна встановити як PWA (програма на робочому столі).

## Запуск

```bash
npm install   # один раз
npm run dev   # http://localhost:5173/
```

## Можливості

### Списки (бічне меню)
- **Вхідні** / **Мій день** (до 5 задач) / **Сьогодні** / **Найближчі** / **Усі задачі** / **Колись** / **Очікування**
- **Огляд тижня** (GTD) / **Матриця** (Ейзенхауер)
- **Журнал** — архів виконаних
- **Таймлайн** — Гант по всіх проєктах
- **Календар** — сітка місяця
- **Статистика** — графік за 7 днів

### Проєкти
- **Області (PARA)** — групування проєктів
- Шаблони, архів, розділи
- Види: Список, Дошка, Таймлайн, Календар, Нотатки

### Задачі
- **Природна мова**: `завтра @офіс 30хв !високий #робота`
- **Відкладення** (defer), **Мій день**, **оцінка часу**
- **Контексти** @дім, @офіс, @телефон…
- **Очікування** (waiting for), **серії** для повторів
- Перетягування, масові дії, шаблони, повторення, нагадування
- Залежності, **Режим фокусу** — Pomodoro 25/5

### Інше
- Скасування видалення (Undo)
- Пошук з фільтрами
- Експорт JSON / CSV / iCal, **імпорт JSON та ICS**
- Автобекап раз на 7 днів, сповіщення (нагадування + Мій день о 8:00)
- Мобільна версія, PWA, тема
- Клавіатура: `Cmd+N/K`, `Cmd+1–5`, `Esc`

## Не в цій версії

- **Синхронізація в реальному часі** між пристроями — лише ручний JSON.
- **Спільна робота в команді** — потребує сервера.
- **Двосторонній sync** з Google/Apple Calendar — лише експорт/імпорт iCal.
